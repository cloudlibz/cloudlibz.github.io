About Us

Cloud is about how you do computing, not where you do computing.

In Cloudlibz, the main focus is to have a central place for all the cloud-related projects of SCoRe (Sustainable Computing Research Group) lab. We provide all the tools necessary to make you cloud savy, meaning that managing your cloud projects is now easier, smarter and more efficient using the tools that we provide. Being an open source organisation, our code base is transparent and anyone who wishes to contribute to our journey towards a smarter cloud savy generation can check out our github page.

WE BELIEVE IN GREAT IDEAS

We've already implemented a number of tools to help you and have various ideas stored in the think-tank, each one coming out of it only when we see potential in it

Nodecloud is a standard library to get a single API among common cloud service providers (Ex. AWS, GCP, Azure.. ). This will make building products among multiple cloud services and its services easier for the developer.

Gocloud is a golang library which hides differences between different cloud providers (AWS,GCP,Openstack etc) APIs and allows you to manage different cloud resources through a unified and easy to use API.
