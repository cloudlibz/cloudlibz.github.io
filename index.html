What is CloudLibz ?

CloudLibz focuses on building products around cloud.
It provides the necessary tools for you to be cloud savvy and makes
managing your projects effortless and easy. We develop products around
unifying cloud environments, tools around cloud automation and
serverless architecture.

Our Products

Nodecloud

NodeCloud is a standard library written in Node.JS. to get a single API among common cloud service providers (Ex. AWS, GCP, Azure.. ). This will make building products among multiple cloud services and its services easier for the developer.

Gocloud

GoCloud is a standard library written in GoLang to get a single API among common cloud service providers (Ex. AWS, GCP, Azure.. ). This will make building products among multiple cloud services and its services easier for the developer.
